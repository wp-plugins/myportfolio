This is a portfolio plugin that provides a animated filtring with popup just with a shortcode that also filters with category.

== Description ==
This plugin provides a simple solution for portfolio filtring and it popups.

  It provides with the shorcode that provides the filtring for the portfolios using isotope filtring.

create portfolios and add categories to it

Use this shortcode to get all the work done. 

[display_portfolio]





== Installation ==
1. Download and install the plugin from WordPress dashboard. You can also upload the entire “myportfolios” folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3.create categories to get filters.
4. add portfolios and select categories.
5. add shotcode to pages or posts [display_portfolio]
